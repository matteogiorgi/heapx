\begin{projectabstract}
    heapx is a compact C99 library and experimental workspace for comparing heap data structures behind one stable public API.
    The project implements a binary min-heap, a classical Fibonacci heap, and the simple Fibonacci heap described by Kaplan, Tarjan, and Zwick.
    The library stores caller-owned \texttt{void *} items, exposes targeted operations through generational handles, and keeps backend-specific details hidden behind an internal dispatch layer.
    This report documents the design, theoretical background, implementation choices, validation strategy, and current benchmark methodology of the project.
    The goal is to keep heapx small enough for experimentation while applying professional engineering practices: explicit ownership rules, overflow-aware allocation helpers, invariant checks, sanitizer runs, reproducible benchmarks, and generated API documentation.
\end{projectabstract}
